\# 🎓 Student Mark Predictor



An interactive Machine Learning web application that predicts student marks based on study hours.



\## 🚀 Features

\- User inputs study hours

\- ML model predicts expected marks

\- Clean and interactive UI

\- Built using Python and Streamlit



\## 🛠 Tech Stack

\- Python

\- NumPy

\- Pandas

\- Scikit-learn

\- Streamlit
